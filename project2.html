Turbocharging Retail Insights

An end-to-end e-commerce data pipeline built on Apache Airflow and Google Cloud Platform that delivers reliable analytics and dashboards for product, marketing, and operations teams.

Repo

Project Overview

This project showcases a robust data pipeline tailored for online retail, built on Apache Airflow and leveraging Google Cloud Platform (GCP) for data storage, processing, and visualization. The pipeline demonstrates automated data ingestion from GCP buckets into BigQuery, robust data quality checks with Soda, transformation using dbt, and dashboards in Metabase.

Key Highlights

  • Automated scheduled ingestion from GCS & external sources to BigQuery.
  • Data quality enforcement via Soda checks integrated as DAG tasks.
  • Transformations modeled with dbt for modular, testable SQL.
  • Actionable dashboards served via Metabase connected to BigQuery.

Technical Stack

  • Orchestration: Apache Airflow (Astro Runtime)
  • Cloud: Google Cloud Platform (GCP)
  • Data Warehouse: BigQuery
  • ETL/Quality: dbt, Soda Core
  • BI: Metabase

Key Features

Automated Data Flow

Effortlessly move data from GCP buckets to BigQuery using Airflow sensors and transfer operators.

Data Quality Assurance

Integrate Soda checks into DAGs to validate schema, row counts, null rates, and custom business rules.

Flexible Transformations

Use dbt models for incremental transformations, tests, and documentation; leverage BigQuery for performance.

Actionable Insights

Metabase dashboards provide product funnels, inventory alerts, and cohort analyses for stakeholders.

Getting Started & Prerequisites

A quick guide to setting up and running the pipeline locally:

# clone the repo
git clone https://github.com/dassubhranil/your-retail-pipeline.git
cd your-retail-pipeline
# setup environment
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt

Tools & Versions

  • Astro CLI: For a user-friendly Apache Airflow interface.
  • Astro Runtime: quay.io/astronomer/astro-runtime:8.8.0
  • Soda Core BigQuery: soda-core-bigquery==3.0.45
  • dbt BigQuery: dbt-bigquery==1.5.3
