Heart Disease Prediction App

An interactive Streamlit web app that predicts heart disease risk using a pre-trained XGBoost model and provides transparent explanations with SHAP.

Live Demo Repo • Built with Streamlit • XGBoost • SHAP

Project Overview

Early detection of heart disease can save lives. This application provides a quick risk assessment using clinical features and explains predictions using SHAP so that clinicians and users can understand which factors drive the model's decision.

Model
XGBoost (XGBClassifier)
Trained on Cleveland heart dataset (303 rows, 13 features)
Explainability
SHAP (TreeExplainer)
Per-prediction SHAP waterfall and feature importances
Interface
Streamlit
Interactive sliders, selects, instant prediction

Features

  • Interactive patient input: 13 clinical features (age, sex, chest pain type, blood pressure, cholesterol, etc.)
  • Real-time risk classification & probability ("Low Risk" / "High Risk")
  • SHAP waterfall & summary plots to explain each prediction
  • Personalized recommendations based on top SHAP drivers
  • Responsive layout for mobile and desktop

Installation & Setup

Run locally with Python 3.9+. It's recommended to use a virtual environment.

Clone & prepare

git clone https://github.com/dassubhranil/heart-disease-app.git
cd heart-disease-app
python -m venv venv
# mac / linux:
source venv/bin/activate
# windows:
# .\venv\Scripts\activate

Install dependencies

pip install -r requirements.txt

# requirements.txt (example)
streamlit
pandas
scikit-learn
xgboost
shap
matplotlib
joblib

Run the App

Launch Streamlit:

streamlit run app.py

By default Streamlit opens at http://localhost:8501. Ensure the model file heart_disease_model.joblib is present in the project root.

FAQ & Notes

What does SHAP tell me?

SHAP assigns an additive contribution to each feature showing how it increases or decreases risk for the prediction. The waterfall plot breaks down the base rate to the final model output.

Is this a medical device?

No — this is a research/demo tool. It is not a replacement for clinical judgement. Always consult a qualified healthcare professional for medical decisions.
